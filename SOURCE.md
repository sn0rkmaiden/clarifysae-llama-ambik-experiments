# Source and provenance

This experiment repository was curated from the user-supplied
`clarifysae-llama-ambik` archive and extended with the concept-vector,
gated-steering, projection, validation, and experiment-planning additions
prepared in this review.

The supplied archive's metadata named Git commit
`0e538aa3cfb8e72ffc910b574452f837a7c0b1d5`. The archive did not include a
`.git` directory, so that commit identity could not be independently verified
from the supplied files.

The original uploaded archives were not modified. Two deliverables are
provided:

1. this compact, source-focused experiment repository; and
2. a full overlay repository that starts from the supplied earlier repository
   and overlays every patched/new file from this repository, preserving the
   earlier historical outputs and visualization assets.
